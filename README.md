# Student Management System

A full-stack Student Management System built with Next.js (App Router), SQLite, and Tailwind CSS.

## Features
- Admin login (cookie-based auth)
- Add / edit / delete / restore students
- Student profile with additional details
- Active / Inactive / Deleted status handling
- Dashboard statistics
- Protected routes using middleware

## Tech Stack
- Next.js 16 (App Router)
- SQLite
- Tailwind CSS
- Node.js

## Setup Instructions

```bash
npm install
npm run dev
